Connecting to the Network

One of the first things that I need to do when I get a computer is to connect to the internet/network. By doing that, I would have access to may more things. Without the network, the user can barely do things except some. Therefore we need to connect ot the internet when we get our computer.

How to connect to the internet

Using a ethernet cable

  1. Plug the wifi modem into a power source
  2. Turn to modem on and wait it to turn on fully
  3. Get a ehternet cable and plug one side into one of the ethernet ports on the modem
  4. Plug the other side of the ethernet cable to the computer's ethernet port
  5. Now the computer is connected to the wifi

Not using a ethernet cable(wireless)

  1. Turn on both the computer and the wifi modem
  2. Go to the wifi settings on the computer
  3. Choose the network you want to use
  4. Type the password if there are any and press connect
  5. Now the computer is connected to the wifi
